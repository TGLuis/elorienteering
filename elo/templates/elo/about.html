About ELOrienteering

Objective

This website aims to do a ranking of the orienteering runners from the Helga Webres results. This ranking is NOT used to for any country selection. This is just a ranking to motivate runners to give their best and to see themselves progress.

What

This is an opensource project ! This means that if you want to contribute, signal an issue or suggest an idea, or just see how this website was made check it on Github.

Disclaimer

This website is still under construction the layout and the data might evolve during the coming weeks. This means that if you are seeing an elo for your name right now, in some days or weeks it might change because some tweaking are still ongoing.
